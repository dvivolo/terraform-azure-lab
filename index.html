Terraform Azure Lab

Infrastructure-as-Code experiments for secure Azure deployments

About This Project

This lab repository contains Terraform configurations for deploying secure resources on Microsoft Azure. It serves as a practical learning project for cloud security, DevSecOps practices, and Infrastructure-as-Code.

Goals

  • Learn Terraform fundamentals and Azure integration.
  • Build reusable Terraform modules with security in mind.
  • Implement secure cloud deployments following least privilege principles.

Planned Repository Structure

terraform-azure-lab/
├── modules/               # Reusable Terraform modules
├── environments/          # Dev / Test / Prod environments
│   └── dev/               # Example environment
│       └── main.tf        # Root configuration file
└── index.html             # Landing page for this repo

Tools Used

  • Terraform (Infrastructure as Code)
  • Microsoft Azure
  • Azure CLI
